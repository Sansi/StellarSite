Introduction

Stellar Wi-Fi uses the WiFi to control Sansi smart bulbs on, off, stepless brightness or color change. The bulb has eight built-in scenarios. You can also interact with the bulb through songs in the library.

Get Application

iOS users may search for "Stellar Wi-Fi" in the App Store.
Android users may search for "Stellar Wi-Fi" in the Google Play Store.
Or you can scan the QR code below.


Install Equipment

Install the LED bulb and turn it on.

iOS Instructions

1. Add device to WiFi

  1. Find the bulb’s Wi-Fi network (name SLRGB*******), and connect to it. Please confirm the connection is successfully established by waiting until the Wi-Fi icon appears at the top of the status bar.
  2. Open the Stellar Wi-Fi application. It will automatically search for the light bulbs, and the connected bulb will be shown in the type list (the bulb is default shown as SSLedLampDefaultName).
  3. Click "click to join the wireless" button, and enter the WiFi name and its password, and then click "join the current wifi". This process takes about 30s.
  4. Having successfully linked to WiFi, please return to the phone settings interface, switch WiFi, and then return to Stellar Wi-Fi. You can find other light bulbs in the list. (If you have refreshed the list many times but couldn't find any bulbs, please try the above steps again).

2. Tourist Mode

The Tourist mode means that Stellar Wi-Fi is used without logging in, and the user can only control bulbs that have been added to the current WiFi. The user enters the mode through the "exit registration" of the sidebar and selects the Tourist mode in the pop-up login interface. The group control function is not available in this mode.

  1. Click the light bulb to enter the control interface. The color control interface shows if it is a RGB bulb, while the color temperature interface shows if it is a CCT bulb.
  2. Music mode (For RGB bulb only)
    Click on the upper left corner of the music selection button, and select your favorite song, and then enjoy colorful lights changing with music.
  3. Scene Mode (For RGB bulb only)
    There are 8 different scene modes. Click to select different scenes, and enjoy colors changing with different frequency.
  4. Countdown Mode (Software version > 2.0 only)
    Click the countdown button on the right side of the switch button on the control panel (If this button is unavailable, please make sure the device software version is the latest one. If not, please update your device software), schedule the time, select the action, click the start button or cancel button.
  5. Config
    In the Config interface, you can do the followings:
    • Change the bulb's name.
    • Switch bulb's WiFi (Only available if the bulb can be found in current WiFi) .
    • If the software version of the bulb is not up to date, please upgrade the firmware. If firmware has updated for a long time, please click Cancel and try again. If you still can not reset the bulb, try to connect the lamp directly. (Only available if the bulb can be found in current WiFi).
    • Reset the lamp (if it is logged in, the bulb will be removed from the list). After reset, the bulb will flash and restored to factory settings.(The bulb will disconnect to the original WiFi).

3. Login Mode

In login mode, the group control and Alexa function are available.

  1. Login
    Click the login button in the side menu, and then enter the account name and password and click login button. After login, Stellar Wi-Fi will automatically find bulbs which have been added into the current WiFi and then add them to the account. If you don't find any bulbs, please refresh the list again.
  2. Connect to Amazon (Alexa)
    Click the "Connect to Amazon" button in the side menu, then click "Confirm the association" button. Follow the Amazon instructions to complete the operation so that you can associate your Stellar account with your Amazon account. Now you can control your bulbs by voice.
  3. Add Group
    Click the button on upper-right corner, then enter the group name and choose bulbs.(Each bulb can only be added to one group. If the bulb you choose is added, it will be removed from the former group).
  4. Group Control
    Click a group to enter the control interface. The group control interface contains color control interface, color temperature control interface and scene control interface.
  5. Edit Group
    Click upper-right button in group control interface to enter Edit Group interface. In this interface, you can change your group's name and add or remove bulbs.

Android Instructions

1. Add device to WiFi

  1. Find the bulb’s Wi-Fi network (name SLRGB*******), and connect it. Please confirm the connection is successfully established by waiting until the Wi-Fi icon appears at the top of the status bar.
  2. Open the app, click the button [+], click [Add device], click [Detected available device], then choose device and click [Save].
  3. Enter the app, click bulb to enter the control interface, click the button on upper right to enter setting interface.
  4. Click [WLAN] to enter a new interface.
  5. Choose the WiFi which you want to join, type the password and then click [Save].
  6. Connect the new WiFi and you can search the bulb.

2. Tourist Mode

The tourist mode means that the user uses Stellar Wi-Fi without logging in, and the user can only control the bulbs that have been added to the current WiFi. The user enters the mode through the "exit registration" of the sidebar and selects the tourist mode in the pop-up login interface.

  1. Click the light bulb to enter the control interface. The color control interface shows if it is a RGB bulb, while the CCT interface shows if it is a color temperature bulb.
  2. Music mode (For RGB bulb only)
    Click on the upper left corner of the music selection button, and select your favorite song, and then enjoy colorful lights changing with music.
  3. Scene Mode (For RGB bulb only)
    There are 8 different scene modes. Click to select different scenes, and enjoy light colors changing with different frequency.
  4. Countdown Mode (Software version > 2.0 only)
    Click the button on the right side of the switch button on the control interface (If you this button is unavailable, please make sure the device software version is the latest one. If not, please update your device software).

3. Login Mode

In login mode, Alexa function is available.

  1. Login
    Type your username, password and click [login].
  2. Connect to Amazon (Alexa)
    Click the [Binding amazon] button in the side menu, then click [Binding amazon] button. Follow the Amazon instructions to complete the operation so that you can associate your Stellar account with your Amazon account. Now you can control your bulbs by voice.
  3. Other skills are the same as Tourist Mode.

Restore Factory Settings

  1. Turn off the bulb and wait for 20 seconds
  2. Repeat the following operations for 3 times.
    1. Turn on the bulb for 5 seconds.
    2. Turn off the bulb for 5 seconds.
  3. Turn on the bulb for the 4th time and wait for 10 seconds. The bulb will now restore the factory settings. It will change colors according to: slow red → slow green → slow blue → warm white and then enter Wi-Fi access point mode.

System Requirements

Stellar Wi-Fi runs on the following devices:

  1. iOS devices running iOS 8.0+
  2. Android devices running Android 4.2+

One or more Sansi Smart LED bulb.
