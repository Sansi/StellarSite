Introduction

Stellar Wi-Fi uses the wireless control Sansi smart bulbs on, off, stepless brightness or choose your favorite color. The bulb has eight built-in scenarios. You can also interact with the bulb through songs in the library.

Get Application

iOS users may search for "Stellar Wi-Fi" in the App Store or scan the first QR code below.

Android users may search for "Stellar Wi-Fi" in the Google Play Store or scan the second QR code below.


Install Equipment

Install the LED bulb and turn it on.

iOS Instructions

1.Add device to router

  1. Find the bulb’s Wi-Fi network (name SLRGB*******), and connect. Please confirm the connection is successfully established by waiting until the Wi-Fi icon appears in the status bar at the top.
  2. Open the Stellar Wi-Fi application. It will automatically search for the light bulbs, and the connected bulb will appear in the class list (the bulb is default shown as SSLedLampDefaultName).
  3. Click "click to join the wireless button", enter the WiFi name and its password, and finally click "join the current wifi", this process takes about 30s.
  4. When the APP prompts you to link successfully, return to the phone settings interface, switch wifi, and then return to Stellar Wi-Fi, you can find in the list to find the lights friends. (If you have refreshed the list many times but haven't find any lights, please try the above steps again).

2.Tourist Mode

The visitor mode means that the user uses Stellar Wi-Fi without logging in, and the user can only control the lights that have been added to the current wifi. The user enters the mode through the "exit registration" of the sidebar and selects the visitor mode in the pop-up login interface, in which the group control function is not available in this mode.

  1. Click the light bulb to enter the control interface. If it is a color light is displayed color control interface, if it is color temperature light shows the color temperature control interface
  2. Music mode (color changed with music, color lights only)
    Click on the upper left corner of the music selection button, select your favorite music, and then feel the beauty which lights shining with the music.
  3. 场景模式（彩色灯特有）
    场景模式为您提供了8种不同的场景模式，点击选择不同场景，感受不同颜色不同频率的渐变。
  4. 倒计时功能（版本>2.0）
    点击控制住界面上开关按钮右边的倒计时按钮（如果没有这个按钮，请检查设备软件版本是否为最新，如果不是，请更新），选择时间，选择动作，点击开始按钮即可启动倒计时，当然也可以点击取消来取消倒计时。
  5. 设置
    在设置界面中，你可以对等进行如下操作：
    • 修改灯的名字；
    • 切换灯的wifi（此功能仅在当前wifi下能找到该灯时可用）；
    • 如果灯的软件版本不是最新的，可以对灯进行固件升级，如果发现更新固件时间较长，请点击取消后重试，如果还是不行，请重置该灯后直接连接灯进行尝试（此功能仅在当前wifi下能找到该灯时可用）；
    • 重置灯（如果处于登录状态，则会将灯从列表中删除），重置后，灯会闪烁，后会恢复出厂设置（脱离原有wifi）。

3.登录模式

登录模式是指用户在登录模式下使用Stellar Wi-Fi,在登录模式下，可以使用组控和Alexa功能

  1. 登录
    在侧边栏点击登录按钮，然后输入账号名及密码点击登录即可（登录后，Stellar Wi-Fi会自动将当前wifi下找到的灯添加到账号下，多刷新几次即可找到灯）。
  2. 连接亚马逊（Alexa功能）
    在侧边栏点击连接亚马逊按钮，后点击确认关联按钮，再根据亚马逊的提示完成操作，即可将你的账号关联至你的亚马逊账号，这要你就可以通过语音来控制你的灯啦。
  3. 添加组
    点击右上角加号，输入组名，选择设备即可添加新的分组（一个设备只能被添加到一个组中，如果选择别的组中的设备，则会将该设备从原有分组中移除）。
  4. 操作组
    点击一个组，进入操作组界面，操作组界面包含颜色控制，色温控制，场景
  5. 编辑组
    在操作界面中点击右上角编辑按钮进入编辑组界面，在编辑组界面中，你可以修改组的名字以及调整组中的设备。

Restore Factory Settings

  1. Turn off the bulb and wait for 20 seconds
  2. Repeat the following operation for 3 times.
    1. Turn on the bulb for 5 seconds.
    2. Turn off the bulb for 5 seconds.
  3. Turn on the bulb for the 4th time and wait for 10 seconds. The bulb will now restore the factory settings. It will change colors according to: slow red → slow green → slow blue → warm white and then enter Wi-Fi access point mode.

System Requirements

Stellar Wi-fi runs on the following devices:

  1. iOS devices running iOS 8.0+
  2. Android devices running Android 4.2+

You also need at least one Sansi Smart LED bulb to get started.
